# Check Course Alignment Workflow

Status: active workflow.

## Uses

- `Current Plan State.md`
- `Work Rails Principles.md`

## Steps

1. Identify the action.
2. Compare it to current focus and plan.
3. Assign tentative K:
   - Kmain
   - Ksupport
   - Klater
   - Kside
   - Koff
4. If unclear, ask for justification.
5. If user saw the course but drifted, mark CV.

## Output

- K rating;
- justification if needed;
- flags if needed;
- next action.
