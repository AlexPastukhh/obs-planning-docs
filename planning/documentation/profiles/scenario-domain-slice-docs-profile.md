# Scenario / Domain / Slice Docs Profile

Status: active reusable specialized profile
Doc version: v0.2.0-obs-cleanup

## Purpose

Use this profile when a project needs scenario/domain/slice documentation boundaries.

## Core Split

```text
Scenario:
  target future/use case and acceptance criteria.

Domain:
  conceptual model and language.

Slice:
  separately deliverable/checkable increment.
```
