# Shared Visibility Map Field Kit

Status: active reusable setup field kit
Doc version: v0.2.0-obs-cleanup
Scope: bootstrap guidance for creating a project-specific shared visibility map if needed.

## Purpose

Use this when local details need to become visible from a shared/root index.

Suggested project output:

```text
planning/shared-visibility-map.md, if the project chooses to create it.
```

## Minimal Rows

```text
- local source;
- shared/root location;
- visibility reason;
- owner;
- update condition.
```

## Do Not

```text
- Do not use this field kit as runtime router after a project map exists.
- Do not copy another project's map as active configuration.
```
