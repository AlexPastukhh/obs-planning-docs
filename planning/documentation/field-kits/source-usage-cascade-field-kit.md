# Source Usage Cascade Field Kit

Status: active reusable setup field kit
Doc version: v0.2.0-obs-cleanup
Scope: bootstrap guidance for project-specific source usage / stale-reference tracking, if needed.

## Purpose

Use this only when a project needs explicit tracking of which docs depend on which sources.

Suggested project output:

```text
planning/source-usage-cascade-profile.md, if the project chooses to create it.
```

## Minimal Row Shape

```text
source file:
used by:
usage type:
stale risk:
review trigger:
```

## Example Policy

Project-specific examples may be kept as demonstration-only. Do not copy example rows as active configuration.

## Do Not

```text
- Do not create source-usage bureaucracy before it is needed.
- Do not use this field kit as runtime router after a project profile exists.
```
