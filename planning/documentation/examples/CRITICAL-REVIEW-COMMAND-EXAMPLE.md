# Critical Review Command Example

Status: supporting reusable example
Doc version: v0.2.0-obs-cleanup
Scope: demonstrates `крит` as review mode.

## Shape

```text
Verdict:
  <ok / risky / blocked / mixed>

Strong points:
  <checked positives>

Weak points / risks:
  <checked risks>

Assumptions:
  <what is not proven>

Next safer action:
  <one concrete action>
```

Boundary:

```text
Do not edit files or create archive unless separately requested.
```
