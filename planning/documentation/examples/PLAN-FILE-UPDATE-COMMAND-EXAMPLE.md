# Plan File Update Command Example

Status: supporting reusable example
Doc version: v0.2.0-obs-cleanup
Scope: demonstrates the `план файл-обновление` command.

## Example Output Shape

```text
Target:
  update planning docs for a named scope.

Changed files:
  - <path>: <what/why>

Not changed:
  - <path>: <why>

Checks:
  - source-of-truth boundary checked;
  - no commit/push permission granted.

Next action:
  produce archive or wait for approval.
```

Owner files:

```text
planning/planning-use-case-map.md
```
